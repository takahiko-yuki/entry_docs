# 前提

- 事前に anyenv で golang をインストールしておく。

# ツールのインストール

- コマンドパレットを開き、ツールをインストールする

```
Go: Install/Update Tools
```

- ツールの候補が表示されるので、選択して OK を押す。
  拘り無ければすべて入れて OK

# デバッガの設定

- ターミナルを開き、デバッガツール dlv が入っていることを確認する

```
dlv version
```

- shift + command + D で実行とデバッグを開き、 launch.json ファイルを作成します　のリンクを選択
